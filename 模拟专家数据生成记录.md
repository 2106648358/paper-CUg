# 模拟专家数据生成记录

## 目的

为论文"AHP-EWM混合权重评估模型"中的**层次分析法（AHP）** 部分提供五位模拟专家的判断矩阵数据，以完整展示多专家整合的流程。

## 方法

采用"优先级向量扰动法"生成每位专家的判断矩阵：

1. **设定每位专家的"真实"权重向量**：基于 IOC 评估框架中六个维度的重要性共识，为每位专家设定一组略有差异的优先级向量，反映不同专业背景的侧重点差异。
2. **构造完全一致矩阵**：由权重向量 $w$ 直接构造 $a_{ij} = w_i / w_j$，得到完全一致的判断矩阵。
3. **添加扰动**：对每个上三角元素乘以对数正态噪声 $\ln\mathcal{N}(0, \sigma)$，$\sigma=0.18$，并四舍五入到最接近的 1-9 标度值，生成合理的非一致性。
4. **一致性检验**：确保每位专家的判断矩阵 $CR < 0.1$。
5. **几何平均整合**：用几何平均法将五位的矩阵整合为综合判断矩阵。
